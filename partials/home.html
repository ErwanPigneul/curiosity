Curiosity

Curiosity is a tool developed by Pages Jaunes, that aims to facilitate datas exploration in elasticsearch. It's fully Open Source (MIT License), developped in javascript with AngularJS.

Our goal is to provide a totally customizable search experience for everyone, with technical skills or not. Curiosity hides elasticsearch's complexity and give to users the possibility to create their dashboards, customize the result list and aggregations, create keywords, export their result in csv... You can with ease almost do whatever you want.

Incoming Features

Right now we are working on the documentation and code comments. There's a lot of things in curiosity, maybe sometimes hard to understand and that's why we try to create a useful and full documentation to this project.

In terms of functionality we will try to implement all elasticsearch search functionality, but before we will work on adding more possibility in templating, to create more interaction between users and curiosity.
