Getting Started

Installation procedure

Curiosity has no external dependency, to use it clone the repository or download the archive from github. Then there is few things that you may have to change in the conf.js file located at the root of Curiosity's folder.The file is like :

//Conf.js

var globalConf = {

defaultServer : 'http://localhost:9200', // Server where curiosity will connect by default

confServer : 'http://localhost:9200', // Server where curiosity will store its data (keywords, template, ...)

confIndex : "curiosity-config", // Index in confServer where curiosity will store it's data

defaultConfDocumentType : "conf-doc", // Type of simple conf document

}

You can modify all values, be sure that you follow json syntax. If you don't, the application will not start properly.

Curiosity is multilingual, right now the application is fully translated in English and French (almost).To change Curiosity's language, copy the file that you need from lang folder to txt.js (This process will be simplified soon).

curiosity/lang> ls

en.js fr.js

curiosity/lang> cp en.js ../txt.js
By default the application is in english (almost).

After that, curiosity is ready. Open index.html with a recent browser and enjoy.

Setting up curiosity's environement

When you open index.html for the first time, curiosity will create its own index and store few conf file. It will take 5 to 10 second (Normally it will take less than 1 second depending of your elasticsearch server, but it's better to wait a bit more).Wait during this period, then reload the page. The page looks like :

At the left of the navigation bar on the top of your screen, their is two buttons that represents tabs. The first button called "Request" is the main tab, and that's where you are. The second button is "Customization", he allow you to edit template and keyword directly in curiosity (go on template and keywords documentation for example).

On the right it's the server selector, you can choose the server you will work on, or add a new one by pressing the "new" button. A form will replace the select, where you can enter the new server address (do not forget the http://), then click the button "Connect" , if it works, the button will stay green and the server will be added to the server list. If not, the button will go red.

The second selector is the indexes and alias selector. You can choose an index, an alias or all indexes to work on.

The light normally green represent curiosity's status, if it goes red, that means an error occurred. If you want more information about the error, check the developper console or curiosity's console(curiosity's console isn't explicit yet, we work on it).

First Query

Curiosity use elasticsearch's query string query syntax, so you can easily create complex query. Click in the query form (first form in page center), enter your query then press "enter" or click on search button. If your request is good then the status light stay green and your result will appear at the bottom center of the page.

Here are some basics for building query :

  • If you want to search a string on a specific field :

    the.path.to.my.fields:"string you search"
